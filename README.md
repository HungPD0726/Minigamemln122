# Quiz Bet Arena

A quiz betting mini-game built with Vite, React, TypeScript, and Tailwind CSS.

## Requirements

- Node.js 20+
- npm 10+

## Run Locally

```bash
npm install
npm run dev
```

App URL: `http://localhost:8080`

## Available Scripts

- `npm run dev`: Start local dev server
- `npm run build`: Build for production
- `npm run preview`: Preview production build locally
- `npm run lint`: Run ESLint
- `npm run test`: Run Vitest unit tests

## Deploy To GitHub Pages

Deployment is automated with GitHub Actions in `.github/workflows/deploy-pages.yml`.
Every push to `main` triggers a new deployment.

Live site: `https://hungpd0726.github.io/Minigamemln122/`
